# Curvenote Preview Theme

A proxy friendly theme server for local previews of Curvenote articles based on a React SPA

## Installation

This theme is automatically installed when used with the Curvenote CLI and added to `curvenote.yml#site.template`.

## Usage

The theme is configured via `template.yml` and can be customized through theme options.
